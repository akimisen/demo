## 设置VSCode默认终端为Bash (3/7/2025, 7:21:39 PM)

_**User**_

如何设置vscode window，默认commandline 我想设置为bash

---

_**Assistant**_

要在 Visual Studio Code 中将默认命令行设置为 Bash，请按照以下步骤操作：

1. 打开 Visual Studio Code。
2. 进入设置，可以通过点击左下角的齿轮图标，然后选择“设置”，或者直接按 `Ctrl + ,`。
3. 在搜索框中输入 `terminal.integrated.shell.windows`。
4. 找到 `Terminal > Integrated: Shell: Windows` 设置项。
5. 将其值更改为 Bash 的路径，例如：
   - 如果你安装了 Git Bash，路径通常是 `C:\Program Files\Git\bin\bash.exe`。
   - 如果你使用的是 Windows Subsystem for Linux (WSL)，可以设置为 `C:\Windows\System32\bash.exe`。

6. 关闭设置窗口，打开新的终端，应该会看到 Bash 作为默认终端。

如果你需要更详细的步骤或有其他问题，请告诉我！
